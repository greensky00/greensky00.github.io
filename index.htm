Personal Projects

  • Generic Skiplist container
    • Skiplist implementation that can be used as set or map, containing any type of data.
    • Completely lock-free for concurrent readers and writers.
    • Scalable throughput according to the number of threads.

  • Partial lock
    • Grab a Lock on a small part of a large shared data, without blocking other locks on non-overlapping parts.

  • C-style generic AVL-tree container
    • An ordered index that can be used as set or map, containing any type of data.
    • Can be used in pure C.
    • Up to 3x faster than STL set or map.
